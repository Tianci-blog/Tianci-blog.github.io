使用 v-md-editor 上传图片到 nodejs(express)

之前有这个需求，顺便记录一下。

前端部分

v-md-editor 中没有构建 form 表单来上传文件，所以我们通过 FormData 对象 来模拟一个表单。向服务器的 http://localhost:3000/upload 接口发送 post 请求。

后端部分

因为只是上传一张图片，所以用到了 multer 的 single 接口, 其参数 avatar 字符串对应客户端表单对象 formData 中包含的 avatar 字段，这两个值必须保持一致。当客户端请求 /upload 接口的时候，multer 会给 request 对象中添加一个 file 对象属性。
